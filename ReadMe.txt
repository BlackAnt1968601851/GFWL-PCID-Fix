Open the GFWL PCID fix exe file and click generate.

(For Developers)
1. Python 3.8.0 is required. For support for windows 7 users.
2. You can run "Compile.bat" to compile the files and it will be inside dist folder
3. If you don't have pyinstaller or Pyperclip on your system run the "Install Redistributables.bat"
make sure you python is in the System Path and the scripts folder inside python too.
4. You can run the source code directly by running "run source code.bat"
5. If you want to edit any of the batch scripts just right click and click edit or click
open in notepad
